GlobalCurve
===========

global curve fitting software


Installation
============
- install globalcurve
- install pgplot

Set system path to recognize the globalcurve executable.


### pgplot
install at c:


Usage
=====

In windows, use PowerShell. To access documents, which are shared using VMWARE, `cd z:`. cd to directory of interest. 

```
> globalcurve
```
